# W4: Live Auto-Capture Gate Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Bot monitors opted-in channels passively and captures feedback automatically. High gate confidence → Status "New" (silent). Medium/low → Status "Needs Review". Eddie marks "Not Feedback" in Notion → webhook archives row and clears dedup.

**Architecture:** Pure pre-filter function (unit-tested) → live Slack `message` event → gate classify → `handleCapture` with `initialStatus`. Parallel: small HTTP webhook server listens for Notion `page.updated` events and handles "Not Feedback" verdicts. All wired in `src/index.ts`.

**Tech Stack:** TypeScript/Node ESM, `tsx`, `@slack/bolt` (already installed), Node built-in `node:http`. No new dependencies.

## Global Constraints

- All imports use `.js` specifiers even for `.ts` source files (ESM NodeNext)
- No build step — run via `tsx`
- Gate fail-open: if `classify()` returns `null`, skip silently (never block)
- Pre-filter is pure — no async, no I/O. Unit-testable.
- `LIVE_GATE_CHANNEL_IDS` — opt-in, comma-separated env var. Empty = live gate disabled everywhere.
- `NOTION_WEBHOOK_PORT` — optional, default 3456. Webhook server only starts if set.
- Model: `claude-haiku-4-5-20251001` — do not change.
- Hosting is out of scope. Webhook tested locally via ngrok.

**Depends on W1 complete:** `DedupStore.delete()`, `DedupStore.findKeyByPageId()`, `FeedbackRecord.status`, `CaptureRequest.initialStatus`, `TriggerType` includes `"live_gate"`, `Config` has `liveGateChannelIds`.

```typescript
// src/liveGate/filter.ts

export interface LiveGateMessage {
  userId?: string;           // undefined = bot/system message
  text: string;
  wordCount: number;
  hasImage: boolean;
  alreadyDuped: boolean;     // true = dedup store already has this message
}

export interface LiveGateFilterOpts {
  botUserId: string;
}

/** Pure one-word regex: matches strings that are only emoji characters. */
const ONLY_EMOJI = /^(\p{Emoji_Presentation}|\p{Extended_Pictographic}|\s)+$/u;

/** Short acknowledgement phrases that carry no product signal. */
const ACK_PHRASES = new Set([
  "sounds good", "thanks", "thanks!", "got it", "noted", "will do",
  "ok", "okay", "perfect", "great", "sure", "yep", "yes", "no", "nope",
  "👍", "👎", "🙏", "✅", "❌",
]);

const MIN_WORDS_WITHOUT_IMAGE = 8;

/**
 * True if the message is worth sending to the feedback gate.
 * Drops: bot messages, very short text-only messages, pure emoji,
 * known acknowledgement phrases, already-captured messages.
 */
export function isLiveGateCandidate(
  msg: LiveGateMessage,
  opts: LiveGateFilterOpts,
): boolean {
  // Bot or system message
  if (!msg.userId || msg.userId === opts.botUserId) return false;

  // Already captured
  if (msg.alreadyDuped) return false;

  // Too short and no image
  if (!msg.hasImage && msg.wordCount < MIN_WORDS_WITHOUT_IMAGE) return false;

  const trimmed = msg.text.trim();

  // Pure emoji
  if (ONLY_EMOJI.test(trimmed)) return false;

  // Acknowledgement phrases (normalise: lowercase, strip punctuation)
  const normalised = trimmed.toLowerCase().replace(/[!?.]+$/, "").trim();
  if (ACK_PHRASES.has(normalised)) return false;

  return true;
}
```

```typescript
const onMessage = async (event: LiveGateEvent): Promise<void> => {
  // Channel must be opted in
  if (!liveGateChannelIds.has(event.channelId)) return;
  if (!liveGate) return;

  const key = `${event.channelId}:${event.messageTs}`;
  const wordCount = event.text.trim().split(/\s+/).filter(Boolean).length;

  // Pre-filter
  if (!isLiveGateCandidate(
    { userId: event.userId, text: event.text, wordCount, hasImage: event.hasImage, alreadyDuped: dedup.has(key) },
    { botUserId: botUserId ?? "" },
  )) return;

  // Gate
  const channelName = await slack.resolveChannelName(event.channelId).catch(() => event.channelId);
  const gateResult = await liveGate.classify(event.text, channelName);
  if (!gateResult || !gateResult.isLikelyFeedback) return;

  // Route by confidence
  const initialStatus: "New" | "Needs Review" =
    gateResult.confidence === "High" ? "New" : "Needs Review";

  const req: import("./core/events.js").CaptureRequest = {
    triggerType: "live_gate",
    channelId: event.channelId,
    messageTs: event.messageTs,
    triggeredBy: event.userId ?? botUserId ?? "",
    initialStatus,
  };

  const result = await handleCapture(req, deps);
  logger.info("Live gate capture", { key, status: result.status, confidence: gateResult.confidence, initialStatus });
};
```

## Task 5: Notion Webhook Server

**Files:**
- Create: `src/adapters/webhook/notionWebhook.ts`

**Interfaces:**
- Consumes: Notion `page.updated` webhook payload; `onNotFeedback(pageId: string) => Promise<void>` callback
- Produces: HTTP server on `NOTION_WEBHOOK_PORT` that calls `onNotFeedback` when `"Gate Verdict"` = `"Not Feedback"`

**Note on Notion webhook payload:** Notion sends a POST to your endpoint with a JSON body. The relevant payload shape for `page.updated`:

```json
{
  "type": "page.updated",
  "entity": { "id": "page-id-here", "type": "page" },
  "data": { "updated_properties": ["Gate Verdict"] }
}
```

However, Notion does not include property values in the webhook — only which properties changed. The handler must re-fetch the page to read the `"Gate Verdict"` select value.

- [ ] **Step 1: Write `src/adapters/webhook/notionWebhook.ts`**

```typescript
// src/adapters/webhook/notionWebhook.ts
import { createServer, type IncomingMessage, type ServerResponse } from "node:http";
import { Client } from "@notionhq/client";
import type { Logger } from "../../util/logger.js";

export interface NotionWebhookDeps {
  notionApiKey: string;
  logger: Logger;
  /** Called when a Notion page's "Gate Verdict" is set to "Not Feedback". */
  onNotFeedback: (pageId: string) => Promise<void>;
}

function readBody(req: IncomingMessage): Promise<string> {
  return new Promise((resolve, reject) => {
    let body = "";
    req.on("data", (chunk) => { body += chunk; });
    req.on("end", () => resolve(body));
    req.on("error", reject);
  });
}

export function startNotionWebhook(port: number, deps: NotionWebhookDeps): void {
  const { notionApiKey, logger, onNotFeedback } = deps;
  const notion = new Client({ auth: notionApiKey, notionVersion: "2022-06-28" });

  const server = createServer(async (req: IncomingMessage, res: ServerResponse) => {
    if (req.method !== "POST") {
      res.writeHead(405).end();
      return;
    }

    res.writeHead(200).end("ok"); // acknowledge immediately (Notion retries on non-2xx)

    try {
      const body = await readBody(req);
      const payload = JSON.parse(body);

      if (payload?.type !== "page.updated") return;
      const pageId: string | undefined = payload?.entity?.id;
      if (!pageId) return;

      // Re-fetch page to read the current Gate Verdict value
      const page: any = await notion.pages.retrieve({ page_id: pageId });
      const verdict: string | undefined = page.properties?.["Gate Verdict"]?.select?.name;

      if (verdict !== "Not Feedback") return;

      logger.info("Notion webhook: Gate Verdict = Not Feedback", { pageId });
      await onNotFeedback(pageId);
    } catch (err) {
      logger.error("Notion webhook handler error", { err: String(err) });
    }
  });

  server.listen(port, () => {
    logger.info(`Notion webhook server listening on port ${port}`);
  });
}
```

```typescript
if (config.notionWebhookPort && config.notionApiKey) {
  const { Client } = await import("@notionhq/client");
  const notionClient = new (Client as any)({ auth: config.notionApiKey, notionVersion: "2022-06-28" });

  startNotionWebhook(config.notionWebhookPort, {
    notionApiKey: config.notionApiKey,
    logger,
    onNotFeedback: async (pageId) => {
      // Archive the Notion page (Notion API: archive = soft delete)

      if (key) {
        dedup.delete(key);
        logger.info("Cleared dedup entry for archived row", { key });
      }
    },
  });
}
```
